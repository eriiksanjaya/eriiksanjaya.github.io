LICENSE IS REQUIRED

Edit the file config/data/licence.js

Right-click the config file and select "Open with Notepad++".

var cfgVal = "masukkan_lisensi_di_sini";

Change it to

var cfgVal = "djy[nkq~ltjUv5iX$dn09";

The license above is a short sample. Please use your actual license sent via email.

Do not delete the quotation marks ("...") and
keep the license in a single line, without pressing Enter.

Other Config Files

To avoid errors, be careful when editing config files. Do not remove quotation marks ("), and commas (,).

Each time you edit a config file, immediately preview the template to check for any errors. It will make troubleshooting easier.

To preview the template, double-click the login.html file

ON / OFF Features

In almost every config file, there is a status setting.

Change the value:

1 = active

0 = inactive

config/data/chat.js - to edit chat settings

config/data/copyRight.js - to edit copyright

config/data/expiredMikhmon.js - to edit expired users in the status page

config/data/header.js - to edit logo, wifi name, and slogan

config/data/internetPackage.js - to edit internet package prices

config/data/licence.js - to edit license information

config/data/login.js - to edit login form

config/data/style.js - to edit styles and colors

Please adjust each config file accordingly.
